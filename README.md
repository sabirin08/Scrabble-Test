# Python Projects
**Focused on building a strong foundation in data structures, algorithms, and problem-solving.**


---

## 📚 Projects

### 🎮 Games & Interactive Programs

* **guess_number.py**
  A number guessing game where users have 10 attempts to guess a randomly generated number between 1 and 100.

* **vending_machine.py**
  Simulates a vending machine with features like purchasing, restocking, and handling invalid selections or out-of-stock items.

* **restaurant_simulator.py**
  Models a restaurant ordering system using dictionaries for menu pricing and calculates total costs based on user input.

---

### 🔤 String & Text Processing

* **reverse_string.py**
  Reverses user-input text using iteration and allows users to exit with a keyword.

* **word_frequency.py**
  Analyzes text input by counting word occurrences and displaying results in a bag-of-words format.

---

### 🔢 Mathematical & Logic-Based Programs

* **scrabbletest.py**
  Calculates the Scrabble point value of a given uppercase word based on tile scoring rules.

* **triangly.py**
  Computes triangle perimeter, area, and classification based on user-provided side lengths.

* **median.py**
  Determines the median of three integer inputs.

---

### ⚙️ Utility & Data Handling

* **step_counter.py**
  Converts steps to miles (1 mile = 2000 steps) with custom exception handling for negative and invalid inputs.

---

### 🧠 OOP Programs

* **gpa_track.py**
  Allows users to build a course roster by adding multiple students and identifying the student with the highest GPA.
  Demonstrates object-oriented programming concepts including classes, objects, methods, input validation, and custom exception handling.

---

### 🗂️ Database Management System
* **database.py**
  Menu-driven Database Management System built in Python that allows users to store, update, retrieve, and delete student and course records.
  Demonstrates foundational concepts in data structures, dictionary-based data storage, and user input handling.

## 🛠️ Skills Demonstrated

* Python Programming
* Control Flow & Loops
* Functions & Modularity
* Exception Handling
* User Input Handling
* Data Structures (Lists, Dictionaries)
* Problem Solving & Logic Development

---

## 🎯 Purpose

These projects were built to strengthen core programming concepts and lay the groundwork for more advanced topics in data structures, algorithms, and software development.
